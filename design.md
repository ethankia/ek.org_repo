# Design Style Guide - Ethan (tragi.c) Developer Portfolio

## Design Philosophy

### Color Palette
- **Primary Background**: Deep charcoal (#0a0a0a) and rich black (#1a1a1a)
- **Accent Colors**: Electric purple (#8b5cf6), neon blue (#3b82f6), cyan (#06b6d4)
- **Text Colors**: Pure white (#ffffff), light gray (#e5e7eb), medium gray (#9ca3af)
- **UV Effect**: Soft purple glow with blue undertones

### Typography
- **Display Font**: "JetBrains Mono" - Modern monospace for code aesthetic
- **Heading Font**: "Inter" - Clean, professional sans-serif
- **Body Font**: "Inter" - Consistent with headings for readability
- **Code Font**: "JetBrains Mono" - Syntax highlighting authenticity

### Visual Language
- **Aesthetic**: Cyberpunk developer noir with professional polish
- **Mood**: Sophisticated, mysterious, cutting-edge
- **Style**: Dark theme with neon accents, code-inspired elements
- **Inspiration**: Terminal interfaces, hacker aesthetic, modern dev tools

## Visual Effects

### Used Libraries
- **Anime.js**: Smooth animations and transitions
- **p5.js**: Mouse-following UV light particle system
- **Shader-park**: Background gradient effects
- **ECharts.js**: Data visualization for project metrics
- **Pixi.js**: Advanced visual effects and code snippet animations
- **Splitting.js**: Text animation effects
- **Typed.js**: Typewriter effects for code snippets

### Effects Implementation
- **UV Mouse Trail**: Canvas-based particle system with purple/blue glow
- **Code Animation**: Typewriter effect with syntax highlighting
- **Background**: Subtle shader-based gradient with noise
- **Text Effects**: Split-letter animations on headings
- **Hover States**: 3D tilt effects with neon glow

### Animation Principles
- **Smooth Transitions**: 300ms ease-in-out for UI elements
- **Staggered Animations**: 100ms delays between elements
- **Particle Physics**: Realistic movement with gravity and friction
- **Code Typing**: Realistic typing speed with variable delays

### Header Effect
- **Navigation**: Fixed top with blur backdrop
- **Logo**: Animated text with glow effect
- **Menu Items**: Hover effects with underline animation
- **Mobile**: Hamburger menu with smooth slide animation

## Layout Structure
- **Grid System**: CSS Grid with Tailwind utilities
- **Responsive**: Mobile-first approach with breakpoints
- **Spacing**: Consistent 8px base unit system
- **Containers**: Max-width with centered alignment

## Interactive Elements
- **Buttons**: Rounded corners with gradient backgrounds
- **Forms**: Dark inputs with neon focus states
- **Cards**: Subtle shadows with hover lift effects
- **Code Blocks**: MacBook-style window with syntax highlighting
