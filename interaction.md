# Interaction Design - Ethan (tragi.c) Developer Portfolio

## Core Interactions

### 1. Dynamic Code Snippet Display
- **Location**: Homepage center
- **Function**: Displays rotating code snippets that change on page load
- **Style**: MacBook-style UI with syntax highlighting
- **Content**: Various programming languages and project examples
- **Animation**: Smooth transitions between code snippets

### 2. Mouse-Following UV Light Effect
- **Location**: Entire website
- **Function**: UV light trail follows mouse cursor
- **Style**: Subtle purple/blue glow effect
- **Implementation**: Canvas-based particle system
- **Interaction**: Dynamic response to mouse movement

### 3. Contact Form with Discord Integration
- **Location**: Contact page
- **Function**: Send form data to Discord webhook
- **Fields**:
  - Name (text input)
  - Discord username (text input)
  - Payment method dropdown (CashApp, PayPal, Apple Pay)
  - Project type dropdown (Website, Mobile App)
  - Understanding confirmation dropdown (Yes/No)
- **Integration**: POST request to Discord webhook
- **Validation**: Required fields and format checking

### 4. Snake Game
- **Location**: Projects page
- **Function**: Classic snake game using arrow keys or WASD
- **Features**:
  - Infinite gameplay
  - Score tracking
  - Smooth movement
  - Food generation
  - Game over/restart functionality
- **Controls**: Arrow keys or WASD
- **Visual**: Retro-style with modern effects

## Navigation Flow
- **Home** → Code display + navigation to pricing/contact
- **Pricing** → Service tiers with clear pricing structure
- **Contact** → Form + Discord server link
- **Projects** → Snake game + coming soon section

## User Experience
- Smooth transitions between pages
- Consistent dark theme with UV effects
- Responsive design for all devices
- Clear call-to-action buttons
- Professional developer aesthetic
